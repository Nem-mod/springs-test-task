- Install python=>3.10 version

## How to run Demo:

- Open file test_task_with_llama-index.ipynb
- Set up an Open AI token
```
import os

OPENAI_API_KEY = "YOUR_API_KEY"
os.environ["OPENAI_API_KEY"] = OPENAI_API_KEY
```
- Run test_task_with_llama-index.ipynb
